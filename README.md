# Out Run TS

- Node.js, TypeScript &amp; the Webpack Dev Server
- Minimum code
- Pseudo 3D 

## 1. Install Node.js packages
```
npm install
```

## 2. Start the Webpack Dev-Server
```
npm run build:devserver
```

## 3. Open the app in your browser
```
http://localhost:1234
```
